# Trader Behavior vs Market Sentiment

This project analyzes the relationship between trader behavior and performance
under different Bitcoin market sentiment conditions using historical Hyperliquid
trading data and the Bitcoin Fear & Greed Index.

## Key Insights
- Traders take larger positions during Greed periods.
- Fear phases show more conservative trading behavior.
- Market sentiment has a measurable impact on profitability and risk-taking.

## Tools
- Python
- Pandas
- Matplotlib

## Notes
Due to the large size of the historical dataset, only relevant columns were used
for analysis.
